    T1. 五一礼物【算法赛】

    劳动创造价值，奋斗成就梦想。正是无数劳动者的辛勤付出，才有了我们今天的美好生活。让我们怀着感恩的心，向每一位劳动者致敬，感谢他们的辛勤付出和无私奉献。

    为了庆祝这个属于劳动者的节日，蓝桥云课特别准备了一份礼物，希望能够带给每一位劳动者、参赛者快乐和祝福。

    请你输出 51，领取这份礼物！

    输入格式

    本题为填空题，无需输入即可作答。

    输出格式

    T2. 合成贤者之石【算法赛】

    问题描述

    炼金术士小蓝毕生梦想是合成贤者之石。

    合成贤者之石需三种水晶：红水晶、蓝水晶以及黄水晶。红水晶能量最强，蓝水晶次之，黄水晶最弱。

    对此，小蓝准备了 nnn 个坩埚来合成贤者之石，第 iii 个坩埚最多容纳 aia_iai​ 颗水晶。为保证能量平衡，每个坩埚需满足：

    • 若坩埚中有红水晶，则红水晶数量 < 蓝水晶数量 < 黄水晶数量。
    • 若坩埚中没有红水晶，但有蓝水晶，则蓝水晶数量 < 黄水晶数量。
    • 若坩埚中只有黄水晶，则无需满足任何条件。

    在合成关键时刻，嫉妒小蓝的黑魔法师小桥，向每个坩埚中放入了若干蓝水晶和黄水晶试图破坏平衡。小桥并不知道能量配比规则，只是随意放入了一些，且蓝水晶数量 < 黄水晶数量，蓝水晶数量 + 黄水晶数量 ≤\leq≤ 坩埚的容量。

    现在，小蓝需尽快调整水晶数量以确保能力平衡。请问，在所有可能的情况中，他最多能向这 nnn 个坩埚中放入多少颗红水晶？

    输入格式

    第一行包含 111 个整数 nnn（1≤n≤1031\leq n \leq 10^31≤n≤103），表示坩埚的数量。

    第二行包含 nnn 个整数 a1,a2,…,ana_1, a_2, \dots, a_na1​,a2​,…,an​（1≤ai≤1051\leq a_i \leq 10^51≤ai​≤105），其中 aia_iai​ 表示第 iii 个坩埚的容量。

    输出格式

    输出一个整数，表示在所有可能的情况中，小蓝最多能向这 nnn 个坩埚中放入的红水晶数量。

    样例输入

    3
1 1 6

    样例输出

    1

    样例说明

    无论如何，小蓝也无法想第 111、222 个坩埚中放入红水晶。

    对于 333 个坩埚，最优情况为：小桥向第 333 个坩埚中放入了 333 个黄水晶、222 个蓝水晶，此时，小蓝可以向其放入 111 个红水晶。

    T3. 咒语融合【算法赛】

    问题描述

    一场盛大的魔法比赛即将举行。

    小蓝，是一位初出茅庐却天赋异禀的法师。为了在比赛中脱颖而出，他翻遍了图书馆中所有积满灰尘的魔法书，终于找到了一种失传已久的组合魔法。这种魔法需要将两个基础咒语融合，从而创造出威力强大的全新咒语。

    每个基础咒语都对应着一个神秘的数字，当两个咒语融合时，它们的数字会相加，得到新咒语的威力值。然而，魔法世界总是充满了奇奇怪怪的规则。如果两个咒语在相同数位上有相同的数字，那么新咒语的威力就会受到削弱，需要减去这些数字。

    例如，如果小蓝选择的咒语是 345345345 和 543543543，那么新咒语的威力值最初为 345+543=888345 + 543 = 888345+543=888。但是，由于 345345345 和 543543543 在十位上都有数字 444，因此新咒语的威力值需要减去 444，最终变为 884884884。

    魔法书中记载了 nnn 个咒语，分别用 {a1,a2,…an}\lbrace a_1, a_2, \dots a_n\rbrace{a1​,a2​,…an​} 表示。小蓝可以从中任意选择两个咒语进行组合。

    请问，小蓝能组合出的咒语的最大威力是多少？

    输入格式

    第一行包含一个整数 nnn（2≤n≤1052\leq n \leq 10^52≤n≤105），表示魔法书中基础咒语的数量。

    第二行包含 nnn 个整数 a1,a2,…,ana_1, a_2, \dots, a_na1​,a2​,…,an​（1≤ai≤1051\leq a_i \leq 10^51≤ai​≤105），表示每个基础咒语对应的数字。

    输出格式

    输出一个整数，表示小蓝能够组合出的咒语的最大威力值。

    样例输入

    3
9 9 8

    样例输出

    17

    具体而言，对于表达式 (a1)op1(a2)op2(a3)...opn−1(an)(a_1) op_1 (a_2) op_2 (a_3)...op_{n-1}(a_n)(a1​)op1​(a2​)op2​(a3​)...opn−1​(an​) ，Yaya 需要在 op1op_1op1​ 到 opn−1op_{n-1}opn−1​ 这 n−1n-1n−1 个位置上填入加号或者减号，加号总数不超过AAA，减号总数不超过BBB，且让表达式的值最大。

    亲爱的见证者啊，这种题目对您来说肯定不在话下。至于 Yaya， 让我们祝她算得出来。

    输入格式

    第一行输入三个整数 n,A,Bn,A,Bn,A,B （1≤A,B≤n≤105,A+B≥n−11 \le A,B \le n \le 10^5 , A+B \ge n-11≤A,B≤n≤105,A+B≥n−1），分别表示序列的长度、加号的数量和减号的数量。

    第二行输入 nnn 个整数 a1,a2,...,ana_{1},a_{2},...,a_{n}a1​,a2​,...,an​ , 表示给定的序列 (−109≤ai≤109-10^9 \le a_i \le 10^9−109≤ai​≤109)。

    输出格式

    输出一个整数，表示可以形成的表达式的最大值。

    样例输入

    说明

    对于样例，我们有 444 个加号，222 个减号。取到最大值 888 时，表达式可以为 1+(−3)−(−5)−(−4)+11 + (-3) - (-5) - (-4) + 11+(−3)−(−5)−(−4)+1 。

    T6. Yaya 与游戏【算法赛】

    问题描述

    到了高中，Yaya 和 Clrlss 被分到了隔壁班。然而，他们仍旧总是在一起玩游戏。虽然，游戏的结果通常是 Clrlss 获胜，但这并没有打消 Yaya 的积极心。

    果然，到了这一天，Yaya 又找到了 Clrlss ，想玩一个新的游戏。游戏的规则是这样的：

    Clrlss 和 Yaya 各自带领了一支队伍进行比拼。Clrlss 一共有 XXX 元钱 , 他的队内有 nnn 个人，第 iii 个人有 aia_iai​ 元钱； Yaya 一共有 YYY 元钱 , 她的队内有 mmm 个人，第 iii 个人有 bib_ibi​ 元钱。

    有无穷多个商店排成一排，编号依次为 1,2,3...1,2,3...1,2,3... 。 其中，编号为 iii 的商店售卖编号为 iii 的彩票，且一张彩票的价格为 iii 元钱 。

    在每一队的所有人各自进行商量之后，对于每一个人，他可以任意在每个商店中购买任意张彩票，但花费的金额不能超过他拥有的钱数，也不能向别人借钱。

    在每个人购买完成之后，两支队伍的所有人共同将他们买到的彩票放在一起。在其中挑出只出现过一次的、编号最小的彩票，拥有这张彩票的人所在的队伍获得游戏的胜利。

    例如，假如 Clrlss 队在商店 1,41,41,4 各买了一张彩票，在商店 222 买了两张彩票；Yaya 队在商店 1,31,31,3 各买了一张彩票，那么最终获胜的是 Yaya 队，因为两队的彩票合起来的编号为 1,1,2,2,3,41,1,2,2,3,41,1,2,2,3,4 ，其中 333 是只出现过一次的、编号最小的彩票，而这张彩票属于 Yaya 队。

    在多年的磨炼下， Clrlss 和 Yaya 都已经绝顶聪明了，他们都可以找到令自己这一队最优的决策。那么，哪一队会赢得最终的胜利呢？

    输入格式

    第一行两个整数 n,Xn,Xn,X ，表示 Clrlss 队内的人数和他拥有的钱数。

    第二行 nnn 个整数 a1,a2,...,ana_{1},a_{2},...,a_{n}a1​,a2​,...,an​ ，表示 Clrlss 队内每个人各自拥有的钱数。

    第三行两个整数 m,Ym,Ym,Y ，表示 Yaya 队内的人数和她拥有的钱数。

    第四行 mmm 个整数 b1,b2,...,bmb_{1},b_{2},...,b_{m}b1​,b2​,...,bm​ ，表示 Yaya 队内每个人各自拥有的钱数。

    输出格式

5 4
3 5

    T9. Yaya 与数数【算法赛】

    问题描述

    在几十年以后的一个晚上， Yaya 和 Clrlss 吃完晚饭，在阳台依偎着，把全部的注意力无私地献给静谧的夜空。几十年的风风雨雨与不可抗的衰老，让他们的视线无法避免地变得浑浊，思维难以抵抗地变得迟钝。
