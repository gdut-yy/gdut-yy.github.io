    # 输出格式

    # T2. 自助餐【算法赛】

    # 问题描述

    食堂最近推出了自助取餐功能，可以通过盘子的形状自动计算费用。你参与到自助计算价格的项目工作中，视觉组的同学已经帮你通过图像识别把盘子图片转换为了字符串，你只需要计算具体的价格即可。

    餐盘的费用如下表所示：

    餐盘种类yuanxingzhengfangxingchangfangxing
    价格123

    接下来一行 nnn 个字符串，用空格隔开，表示不同的盘子。保证每个字符串都是题目描述中提到的六种之一。

yuanxing zhengfangxing changfangxing sanjiaoxing changfangxing

    # T3. 玩游戏【算法赛】

    # 问题描述

    小 A 和小 B 两个人在海边找到了 nnn 个石子，准备开始进行一些游戏，具体规则如下：小 B 首先将 nnn 个石子分成若干堆，接下来从小 A 开始小 A 和小 B 轮流取石子，每次可以任选一堆石子取走任意个，不可不取，没石子可取的输。问在最优策略的情况下，小 A 和小 B 到底谁能赢得游戏。

    # 输入格式

    一行一个整数 nnn，表示石子个数。

    # 输出格式

    一行一个字符 A 或者 B，输出 A 表示小 A 能赢得游戏，输出 B 表示小 B 能赢得游戏。

    # 样例输入

    2

    # 样例输出

    B

    # 说明

    对于 222 个石子，小 B 将其分成两堆，每堆 111 个石子即可获胜。

    对于 252525% 的评测数据，0<n≤100 < n \leq 100<n≤10。

    # T8. 缺页异常 1【算法赛】

    # 问题描述

    你有一台拥有 nnn 个缓存页面空间的服务器，有 kkk 个用户在同时使用。你预知了这 kkk 个用户将会一共给出 mmm 条页面请求。每条请求以 x,yx, yx,y 的形式表示，表示编号为 xxx 的用户请求了页面 yyy。这里不同用户的相同编号页面认为是不同的。你需要在开始给每个用户分配一个固定的缓存页面空间大小 psips_ipsi​，满足 ∑i=1kpsi=n\sum_{i=1}^{k}{ps_i} = n∑i=1k​psi​=n，并且使用 OPT 页面置换算法，使得总的缺页中断次数最小。

    OPT 算法的具体描述为：每次当需要访问一个页面的时候，假设当前用户持有 nin_ini​ 个缓存空间，首先查询该页面是否存在 nin_ini​ 个缓存页面空间中，若在，则结束；若不存在，则将该页面置入 nin_ini​ 个位置的其中一个（若 ni=0n_i=0ni​=0则不操作），并触发一次缺页中断。

    置入位置的选择规则如下：如果当前空间中有空的位置（为初始状态，历史未在这个位置插入过任何页面），则选择任意一个空位置插入；否则选择当前在缓存中并且未来最久才会被访问到的页面（未来不被再访问的认为无穷久，所如果有多个，任选一个），在该页面的位置插入新页面。一个用户的初始缓存页面空间的每个位置均处在初始状态。

    第一行三个整数，n,k,mn,k,mn,k,m，表示缓存页面空间，用户数，页面请求条数。

    接下来 mmm 行，每行两个整数 x,yx, yx,y 表示一个请求。其中 1≤x≤k1 \leq x \leq k1≤x≤k。

    TTT 行，对应 TTT 组数据的答案，每一行一个整数，表示最小缺页次数。

    # T9. 缺页异常 2【算法赛】

    # 问题描述

    你有一台拥有 nnn 个缓存页面空间的服务器，只有 111 个用户在同时使用。这个用户将会一共给出 mmm 条页面请求。每条请求用一个整数 yyy 表示请求了页面 yyy。请你计算出对于 n∈[0,m]n \in [0, m]n∈[0,m]，使用 LRU 页面置换算法的缺页中断次数是多少。

    LRU 算法的具体描述为：每次当需要访问一个页面的时候，假设当前用户持有 nin_ini​ 个缓存空间，首先查询该页面是否存在 nin_ini​ 个缓存页面空间中，若在，则更新该页面访问时间并结束；若不存在，则将该页面置入 nin_ini​ 个位置的其中一个，并记录当前访问时间，并触发一次缺页中断。

    置入位置的选择规则如下：如果当前空间中有空的位置（为初始状态，历史未在这个位置插入过任何页面），则选择任意一个空位置插入；否则选择当前在缓存中并且之前最久未被访问过的页面（记录的访问时间距离现在最久），在该页面的位置插入新页面。用户的初始缓存页面空间的每个位置均处在初始状态。

    # 输入格式

    第一行一个整数，mmm，表示页面请求条数。

    接下一行，mmm 个整数，表示 mmm 个页面请求的编号。

    # 输出格式

    一行，m+1m+1m+1 个整数，表示当缓存大小为 [0,m][0, m][0,m] 时的答案。

    # 样例输入

    6

    # 样例输出

    当缓存大小为 222 时的执行过程如下： 初始：[] 访问：[1] （缺页） 访问：[3, 1] （缺页） 访问：[1, 3] 访问：[2, 1] （缺页） 访问：[1, 2] 访问：[1, 2] 故缺页中断次数为 333 次。
