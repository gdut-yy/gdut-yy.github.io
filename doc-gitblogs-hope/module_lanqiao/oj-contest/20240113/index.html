    T1. 召唤神坤【算法赛】

    问题描述

    现有 NNN 个帅气逼人的偶像练习生成员，每个成员都有其舞力值 WiW_iWi​。我们需要选出三名成员 i,j,ki,j,ki,j,k (i<j<k)(i <j<k)(i<j<k) 组成舞团出道，舞团的舞力值计算规则为 ⌊Wi+WkWj⌋\lfloor \dfrac{W_i+W_k}{W_j} \rfloor⌊Wj​Wi​+Wk​​⌋，其中 ⌊⌋\lfloor \rfloor⌊⌋ 表示下取整。

    当成功选出舞力值最大的团体组合时，就可以召唤神坤，请你计算出可能选出的最大团队舞力值。

    输入格式

    第一行输入一个正整数 NNN 表示偶像练习生成员个数。

    第二行输入 NNN 个整数 W1,W2,W3,⋯WnW_1,W_2,W_3, \cdots W_nW1​,W2​,W3​,⋯Wn​ 表示成员的舞力值。

    输出格式

    输出一个整数，表示可能选出的最大团队舞力值。

    样例输入

    6
1 2 3 4 5 6

    样例输出

    3

    说明

    一种可能的选择方案为选择 [1,2,6][1,2,6][1,2,6]，可以得到最大团队舞力值。

    评测数据范围

    3≤N≤1053 \leq N \leq 10^53≤N≤105，1≤ai≤1091 \leq a_i \leq 10^91≤ai​≤109。

    他们轮流攻击对方，小蓝先进行攻击，然后是小桥，然后轮到小蓝，以此类推。

    每一次进行攻击，被攻击的一方都会减少体力值，减少的体力值等于攻击方的攻击力，当某一方的体力值小于等于 000，那么游戏结束，体力值先小于等于 000 的一方输。

    小蓝是一个计算机高手，他知道了小桥的体力值和攻击力，他决定调整自己的攻击力，以保证自己一定能赢，但是他又不想太明显，以至于被小桥发现，所以他决定调整自己的攻击力为保证自己一定能赢的最小值。

    输入格式

    输入包含多个测试组。

    输出格式

    输出 TTT 行，每行一个值，表示小蓝能赢的最小攻击力。

    3
5 6 1
10 16 10
8 8 3

    样例输出

    2
16
3

    说明

    第一组样例，如果小蓝的攻击力为 111，那么在轮流攻击 555 次后，就会输掉，所以最小是 222。

    第二组样例，小蓝必须一击必杀。

    为了防止滑冰场内太冷，滑冰场的老板小蓝在 ttt 个位置放置了暖气炉，由于蓝桥冰场的冰是特殊材质，所以你并不用担心冰会融化。

    每个暖气炉会产生暖气。在放置时，放置的位置就已经被暖气覆盖。每经过一秒，存在暖气的格子会传播暖气到周围的格子。具体来说，如果 (x,y)(x, y)(x,y) 存在暖气，那么一秒之后，(x−1,y),(x+1,y),(x,y−1),(x,y+1),(x−1,y−1),(x−1,y+1),(x+1,y−1),(x+1,y+1)(x-1, y),(x+1, y),(x, y-1),(x, y+1),(x-1, y-1),(x-1, y+1),(x+1, y-1),(x+1, y+1)(x−1,y),(x+1,y),(x,y−1),(x,y+1),(x−1,y−1),(x−1,y+1),(x+1,y−1),(x+1,y+1) 八个位置也会被暖气覆盖。

    T8. 逃跑【算法赛】

    每一秒，两人都能移动到他们当前位置相邻的节点，或者也可以选择不移动。当小蓝和小桥处于同一个位置时，可以认为小桥抓住了小蓝。

    注意：两人的移动是线性的，他们会沿着边等速移动，所以他们也可能在路中间相遇。

    小蓝，小桥都是很聪明的，无论出现在哪一个点，小蓝总会逃跑尽可能长的时间，以坚持最长的时间不被小桥抓住，而小桥也会尽可能以最短的时间抓出小蓝。

    如果答案是 ab\frac {a}{b}ba​ 的形式，你需要输出 a×b−1mod  998244353a \times b^{-1} \mod 998244353a×b−1mod998244353，b−1b^{-1}b−1 是 bbb 在模 998244353998244353998244353 下的逆元，题目保证答案一定存在。

1 2
1 3
3 4

    3

    说明

    1. 出现在 111 点，最晚 000 秒时被抓住。
    2. 出现在 222 点，最晚 111 秒时被抓住。
    3. 出现在 333 点，最晚 444 秒时被抓住。
    4. 出现在 444 点，最晚 444 秒时被抓住。

    T9. 选糖果【算法赛】

    问题描述

    小桥最终还是抓到了小蓝，但是他还是有一颗仁慈之心，他给了小蓝一次机会。

    只要能够回答这个问题，那么小桥就不会给小蓝一万道数学题来折磨小蓝，而是会大发慈悲放他一马。

    问题是这样的，小桥拿出了 nnn 颗糖果，编号 1∼n1 \sim n1∼n。每颗糖果都有自己的种类，共有 kkk 种不同的种类，编号 1∼k1 \sim k1∼k，小桥规定第 iii 种糖果不能取超过 sis_isi​ 颗。每次小桥询问一个区间 [l,r][l, r][l,r]，代表小蓝需要在糖果编号在 [l,r][l, r][l,r] 的区间中取糖果，请问小蓝最多能取多少颗糖果。

    为了验证小蓝不是蒙出来的答案，小桥会询问 mmm 次。

    小蓝当然不会，于是又来求助你了。你需要回答小桥的所有问题。

    输入格式
