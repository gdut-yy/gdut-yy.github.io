    小桥想让小蓝回答她在操作完成后 SSS 是多少？小蓝陷入了困境，于是请你帮帮他！

    # 输出格式

    输出一个字符串表示答案。

    # 输入样例

    5 3
abcde
1 5 3
1 2 4
2 5 3

    # 输出样例

    hlijk

    # T7. 贝贝的集合【算法赛】

    # 问题描述

    贝贝有一个大小为 nnn 的可重集合 {2a1,2a2,⋯ ,2an}\{2^{a_1},2^{a_2},\cdots,2^{a_n}\}{2a1​,2a2​,⋯,2an​}，

    • 可重集合，即集合中的数允许出现多次。

    约定一次操作的过程如下：

    • 第一步，从集合中取出两个数并从集合中删除，约定较小的为 xxx，较大的为 yyy，即 x≤yx\le yx≤y；
    • 第二步，将 2x2x2x 插入集合中；
    • 第三步，若 y exy\ e xy ex，则将 y−xy-xy−x 插入集合中。

    贝贝想知道，经过若干次操作后，集合的大小最小可以为多少？

    # 输入格式

    第一行，包含一个整数 n(1≤n≤105)n(1\le n\le 10^5)n(1≤n≤105)。

    第二行，包含 nnn 个整数 a1,a2,⋯ ,an(1≤ai≤109)a_1,a_2,\cdots,a_n(1\le a_i\le 10^9)a1​,a2​,⋯,an​(1≤ai​≤109)。

    # 输出格式

    仅一行，包含一个整数，表示经过若干次操作后，集合的大小的最小值。

    # 样例输入

    5
1 1 1 3 1

    # 样例输出

    1

    # 说明

    一种贝贝可能的操作顺序如下：

    第一次操作，取出 21,212^1,2^121,21，操作完毕后集合变成 {21,21,22,23}\{2^1,2^1,2^2,2^3\}{21,21,22,23}。

    第二次操作，取出 21,212^1,2^121,21，操作完毕后集合变成 {22,22,23}\{2^2,2^2,2^3\}{22,22,23}。

    第三次操作，取出 22,222^2,2^222,22，操作完毕后集合变成 {23,23}\{2^3,2^3\}{23,23}。

    第四次操作，取出 23,232^3,2^323,23，操作完毕后集合变成 {24}\{2^4\}{24}。

    故最终集合大小的最小值为 111。
