    # T1. 蓝桥小课堂-平方和【算法赛】

    # 问题描述

    蓝桥小课堂开课啦！

    平方和公式是一种用于计算连续整数的平方和的数学公式。它可以帮助我们快速求解从 111 到 nnn 的整数的平方和，其中 nnn 是一个正整数。

    平方和公式的表达式如下：

    12+22+32+…+n2=n⋅(n+1)⋅(2n+1)6 1^2 + 2^2 + 3^2 + \ldots + n^2 = \frac{{n \cdot (n + 1) \cdot (2n + 1)}}{6} 12+22+32+…+n2=6n⋅(n+1)⋅(2n+1)​

    这个公式可以简化计算过程，避免逐个计算整数的平方并相加。通过直接使用平方和公式，我们可以更高效地求解大规模的平方和问题。

    举个例子，如果要计算从 111 到 101010 的整数的平方和，可以将 nnn 的值设为 101010 ，然后将其代入公式中计算：

    12+22+32+…+102=10×(10+1)×(2×10+1)6 1^2 + 2^2 + 3^2 + \ldots + 10^2 = \frac{{10 \times (10 + 1) \times (2 \times 10 + 1)}}{6} 12+22+32+…+102=610×(10+1)×(2×10+1)​

    现在，学习完平方和公式的你需要接受小蓝的考验了。小蓝会给定一个整数 nnn，请你求出 ∑i=1ni2\sum_{i=1}^{n} i^2∑i=1n​i2 的值。

    # 输入格式

    输入一行包括一个整数 nnn。

    # 输出格式

    输出一个整数表示答案。

    # 样例输入

    10

    # 样例输出

    385

    # 评测数据范围

    1≤n≤1061 \leq n \leq 10^61≤n≤106。

    也就是说，假设每个格子有 101010 种情况（X 或者 0∼80 \sim 80∼8），那么 3×m3 \times m3×m 的地图总共有 103m10^{3m}103m 情况，请你找出满足以下两个要求的地图数量：

    1. 地图合法，即满足扫雷规则。
    2. 第二行与给定第二行的值相同。

    答案可能很大，请对 998244353998244353998244353 取模。

    # 输入格式

    第一行输入一个整数 mmm。

    第二行输入 mmm 个整数 p1,p2,p3,...,pmp_1, p_2, p_3, ..., p_mp1​,p2​,p3​,...,pm​，代表第地图中 222 行第 iii 列的值。

    # 输出格式

01X  XX1  000  XXX  1X1  X2X  X10  1XX

    # T9. 香煎七十二餐馆【算法赛】

    # 问题描述

    小蓝开了一家餐馆：“香煎七十二餐馆！”

    据说共有七十二道菜，每道菜煎炸的次数不一，最简单的只需要煎一次，最复杂的需要煎炸整整七十二次！

    开店第一天，举办了一场餐馆派对，免费接待 nnn 位顾客。

    nnn 位顾客坐在一个长桌上，编号 1∼n1 \sim n1∼n。小蓝的餐馆共有 727272 道菜，编号 1∼721 \sim 721∼72。每位顾客只会点一道菜，nnn 位顾客中有 mmm 位顾客已经点好了菜，剩下的顾客由小蓝推荐点菜。

    由于每位顾客用餐的时候，都会对菜品评头论足，如果两个相邻的顾客吃的是同一道菜，他们难免会讨论，有一些吹毛求疵的顾客可能借此将消极情绪传播给他人。为了避免上述情况发生，导致影响口碑，小蓝将安排相邻的顾客的菜肴都不相同。

    请问有多少种不同的安排方案，答案可能很大，请对 998244353998244353998244353 取模。

    相邻：即编号相邻，iii 与 i−1i-1i−1 和 i+1i+1i+1 相邻。注意，这是一张长桌，所以 111 和 nnn 并不相邻。

    # 输入格式

    第一行输入两个整数 n,mn,mn,m。

    接下来 mmm 行，每行两个整数 pi,cip_i, c_ipi​,ci​，表示编号为 pip_ipi​ 的人已经点好了一份编号为 cic_ici​ 的菜肴。

    # 输出格式

    输出一个整数，表示合法的安排方案数。答案可能很大，请对 998244353998244353998244353 取模。

    # 样例输入

    3 1
2 1

    # 样例输出

    5041

    # 说明

    第一个顾客可以安排编号 2∼722 \sim 722∼72 的菜肴，第三位顾客可以安排编号 2∼722 \sim 722∼72 的菜肴。共计 712=504171^2=5041712=5041。

    # 评测数据范围

    0≤m≤105,3≤n≤1015,1≤ci≤72,1≤pi≤n0 \le m\le 10^5, 3 \le n \le 10^{15}, 1 \le c_i \le 72, 1 \le p_i \le n0≤m≤105,3≤n≤1015,1≤ci​≤72,1≤pi​≤n。

    保证所有的 pip_ipi​ 不相同。
